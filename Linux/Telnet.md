## telnet이란?
네트워크 관리를 할 수 있는 프로토콜(원격접속 서비스)

## 시작하기 전에
+ 설치 유무 확인하기  
```
dpkg -s telnetd
```
+ 설치하기
```
sudo apt-get install telnetd
```
+ telnet 클라이언트 설치하기
프로그램 추가/제거 -> windows 기능 켜기/끄기 -> 텔넷 클라이언트 체크
