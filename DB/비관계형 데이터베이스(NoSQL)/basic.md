## NOSQL
: 키-밸류나 컬럼, 문서 형식의 데이터 모델을 이용한다.
(RDBMS가 아닌 다른 형태로 데이터를 저장하는 기술이라고 생각하면 된다.)
NOSQL: Not Only SQL, 비관계형 데이터베이스
* 그렇다면 왜, 언제 NOSQL을 쓰는 걸까? 아주 많은 양의 데이터를 효율적으로 처리가 필요할 때, 데이터의 분산처리, 빠른 쓰기 및 데이터의 안정성이 필요할 때 사용한다
