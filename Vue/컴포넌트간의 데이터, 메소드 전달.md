# 1. vuex 사용
### vue
```js
methods: {
  ...mapState(['anlaysis/A']),
  ...mapMutations(['analysis/setA'}]
}
// 데이터 값 setting
this.$store.commit('analysis/setA', true)
// vuex에서 공유중인 데이터 값 사용
this.A
```
### js
```js
const analysis = {
  state: {
    A: ''
  },
  mutations: {
    setA: (state,data) => {
    state.A = data
  }
```
